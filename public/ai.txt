# Learn With Andi (LWA) - Project Context for AI Agents

## Project Overview
Learn With Andi (LWA) is a career coaching platform helping Indonesian professionals, fresh graduates, and career switchers navigate the job market. The site offers mentoring, CV optimization, LinkedIn branding, and interview preparation strategies.

## Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **UI Styling**: CSS Modules (Vanilla CSS variables)
- **Icons**: Lucide React
- **Deployment**: Vercel

## Core Values & Tone
- **Tone**: Professional yet approachable, "Realistis" (Realistic), encouraging but ground-level (no "get rich quick").
- **Audience**: Job seekers in Indonesia (Fresh Grads, Career Switchers, Professionals).

## Key Data Source
- `src/data/content.ts`: This file contains ALL text content, questions, links, and configuration for the landing page. Edit this file to change website copy.
